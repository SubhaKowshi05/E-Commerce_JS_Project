# E-Commerce_JS_Project
FEATURES
-Object-Oriented Cart implementation using JavaScript classes
-Multiple cart instances (regular cart & business cart)
-Persistent cart storage using LocalStorage
-Dynamic delivery option selection with updated delivery dates
-Real-time price, shipping, tax, and total calculations
-Modular ES6 imports/exports
-Clean separation between data, logic, and UI rendering

TECH STACK
-JavaScript (ES6+)
-HTML5
-CSS3
-LocalStorage API
-Day.js (for delivery date calculations)
-Modular Architecture (ES Modules)

LEARNING OUTCOMES
-Implementing Object-Oriented Programming in JavaScript
-Managing shared state using a single source of truth
-Using ES6 modules for scalable code structure
-Handling DOM events and UI re-rendering efficiently
-Persisting application state using LocalStorage
-Debugging import/export and data flow issues in modular JS
